# GitHub README Generator For Claude Code

Follow `github-readme-generator/SKILL.md` to create polished GitHub README.md files. Load `github-readme-generator/references/readme-playbook.md` for deeper README strategy, `github-readme-generator/references/badges-and-stats.md` when adding Shields.io badges or GitHub Stats Extended cards, and `github-readme-generator/references/resource-catalog.md` when choosing external README tools or visual resources.

When a first draft is useful, run the renderer in `github-readme-generator/scripts/`, then revise the output using repository context. Preserve accurate existing project details, add sections only when they help the reader act, and remove stale claims.
