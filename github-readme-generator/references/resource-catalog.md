# Resource Catalog

Use these resources when improving visual polish, README structure, or automation. Prefer stable, useful resources over decorative clutter.

## Official GitHub References

- [About READMEs](https://docs.github.com/en/repositories/managing-your-repositorys-settings-and-features/customizing-your-repository/about-readmes) - What repository READMEs should communicate.
- [Basic writing and formatting syntax](https://docs.github.com/en/get-started/writing-on-github/getting-started-with-writing-and-formatting-on-github/basic-writing-and-formatting-syntax) - GitHub Flavored Markdown syntax.
- [Quickstart for writing on GitHub](https://docs.github.com/en/get-started/writing-on-github/getting-started-with-writing-and-formatting-on-github/quickstart-for-writing-on-github) - Markdown walkthrough using a profile README.
- [Managing your profile README](https://docs.github.com/en/account-and-profile/how-tos/profile-customization/managing-your-profile-readme) - How profile READMEs work.
- [Community profile docs](https://docs.github.com/en/repositories/managing-your-repositorys-settings-and-features/customizing-your-repository/about-community-profiles-for-public-repositories) - License, code of conduct, and contributing files.

## Structure And Templates

- [Awesome README](https://github.com/matiassingers/awesome-readme) - Examples and articles about strong project READMEs.
- [Standard Readme](https://github.com/RichardLitt/standard-readme) - Predictable open source README specification.
- [Best README Template](https://github.com/othneildrew/Best-README-Template) - Detailed project README template.
- [Make a README](https://www.makeareadme.com/) - Concise guide to useful READMEs.
- [readme.so](https://readme.so/) - Visual README editor.
- [README MD Generator](https://github.com/kefranabg/readme-md-generator) - CLI generator for project README files.

## Profile README

- [Awesome GitHub Profile README](https://github.com/abhisheknaiidu/awesome-github-profile-readme) - Profile README gallery.
- [GitHub Profile README Generator](https://github.com/rahuldkjain/github-profile-readme-generator) - Form-based profile generator.
- [GitHub Profilinator](https://github.com/rishavanand/github-profilinator) - Drag-and-drop profile builder.
- [Profile README Generator](https://profile-readme-generator.com/) - Profile links, skills, and dynamic cards.

## Badges, Cards, And Widgets

- [Shields.io](https://shields.io/) - Custom badges for builds, versions, coverage, downloads, and labels.
- [Badgen](https://badgen.net/) - Compact badge service.
- [Simple Icons](https://simpleicons.org/) - Brand SVG icons and colors.
- [Markdown Badges](https://github.com/Ileriayo/markdown-badges) - Copy-paste tech badges.
- [Skill Icons](https://github.com/tandpfun/skill-icons) - Skill icon strips.
- [GitHub Stats Extended](https://github.com/stats-organization/github-stats-extended) - Stats, languages, pinned repos, gists, and WakaTime cards.
- [GitHub Readme Streak Stats](https://github.com/DenverCoder1/github-readme-streak-stats) - Contribution streak card.
- [GitHub Profile Trophy](https://github.com/ryo-ma/github-profile-trophy) - Trophy-style achievements.
- [Metrics](https://github.com/lowlighter/metrics) - Configurable profile metrics.

## Visuals, Media, And Automation

- [Capsule Render](https://github.com/kyechan99/capsule-render) - Dynamic SVG headers.
- [Readme Typing SVG](https://github.com/DenverCoder1/readme-typing-svg) - Animated profile/tagline SVG.
- [Mermaid](https://mermaid.js.org/) - GitHub-supported diagrams.
- [Excalidraw](https://excalidraw.com/) - Hand-drawn style diagrams.
- [draw.io / diagrams.net](https://www.diagrams.net/) - General diagramming.
- [asciinema](https://asciinema.org/) - Terminal recordings.
- [VHS](https://github.com/charmbracelet/vhs) - Scripted terminal GIF/video recorder.
- [Kap](https://getkap.co/) - Screen recording.
- [ImageOptim](https://imageoptim.com/) and [Squoosh](https://squoosh.app/) - Image compression.
- [Blog Post Workflow](https://github.com/gautamkrishnar/blog-post-workflow), [Waka Readme](https://github.com/athul/waka-readme), and [All Contributors](https://allcontributors.org/) - README automation.
