# GitHub README Generator For Claude Code

Follow `SKILL.md` in this folder to create polished GitHub README.md files. Load `references/readme-playbook.md` for deeper README strategy, `references/badges-and-stats.md` when adding Shields.io badges or GitHub Stats Extended cards, and `references/resource-catalog.md` when choosing external README tools or visual resources.

When a first draft is useful, run the renderer in `scripts/`, then revise the output using repository context. Preserve accurate existing project details, add sections only when they help the reader act, and remove stale claims.
